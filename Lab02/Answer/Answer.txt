1. If you create a constructor method to build a DVD by title then create a constructor method to build a DVD by category. Does JAVA allow you to do this?
- Không thể có DigitalVideoDisc(String title) DigitalVideoDisc(String category) Vì Java chỉ nhìn kiểu dữ liệu, không nhìn ý nghĩa biến
- Cách sửa: 
public static DigitalVideoDisc createByTitle(String title) {
    DigitalVideoDisc dvd = new DigitalVideoDisc(title);
    return dvd;
}

public static DigitalVideoDisc createByCategory(String category) {
    DigitalVideoDisc dvd = new DigitalVideoDisc(null, category, null, 0, 0);
    return dvd;
}
